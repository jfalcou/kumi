Detailed Description

template<typename T, template< typename... > class Meta = std::type_identity>
struct kumi::as_tuple< T, Meta >

Generate a kumi::tuple type from a type.

If T is a kumi::product_type, returns the kumi::tuple type containing the same element as T. Otherwise, it returns kumi::tuple<T>.

A template meta-function can be optionally passed to be applied to each of those types when types are computed.

Template Parameters
TType to transform
MetaUnary template meta-function to apply to each types. Defaults to std::type_identity
